住宅宿泊事业申请流程

流程顺序以“先排除阻断项、再准备届出”为原则；当地规则可能增加事前周知和附件。

阶段1：物件与制度初筛

  1. 确认权利、管理规约、地址和实际建筑。
  2. 核验住宅设备与居住要件。
  3. 确认区条例、用途地域和实施期间。
  4. 判断家主居住型或家主不在型。

阶段2：事前咨询

  1. 预约主管自治体或保健所。
  2. 携图纸咨询消防署。
  3. 需要时咨询建筑、清扫和都市计划部门。
  4. 确定管理委托和紧急响应。

阶段3：届出前程序

  1. 按当地范围和提前日数完成近邻说明。
  2. 签订垃圾回收、管理和必要服务合同。
  3. 整理登记录、合同、规约、图纸和誓约资料。
  4. 完成安全措施清单。

阶段4：届出与开业

  1. 通过民泊制度运营系统制作和发送届出。
  2. 按当地方式提交附件并完成人员确认。
  3. 处理补正，取得届出编号和标识。
  4. 确认标识掲示后再开始营业。

阶段5：开业后

  • 宿泊者名簿、本人确认、卫生安全和多语言说明
  • 垃圾、噪音、投诉和紧急响应
  • 每两个月定期报告
  • 管理业者、法人、住所、设施等事项变更时办理手续
