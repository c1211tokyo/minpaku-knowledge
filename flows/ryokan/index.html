旅馆业许可申请流程

每阶段都有停止条件；不满足时回到资料、设计或制度选择，而不是继续投入。

0. 目标定义

明确营业类别、房型、人数、前台或替代管理、预计开业日和预算上限。

1. 投资前初筛

  1. 权利、租赁、规约和施工权限。
  2. 用途地域、现况用途、用途变更和违法增建。
  3. 整栋消防用途、设备和避难。
  4. 当地条例、学校或近邻程序。

停止：权利不清、用途明显不允许、合法图纸无法取得或高成本消防无法接受。

2. 多部门事前咨询

  1. 保健所确认类别、结构设备、卫生和资料。
  2. 消防署确认用途、设备、届出和检查。
  3. 建筑部门确认用途、变更、现况和接道。
  4. 需要时咨询清扫、都市计划和近邻主管部门。

证据：咨询日期、部门、问题、答复、条件、资料版本和下一步。

3. 设计、预算和专业确认

由建筑士等把各部门条件合并成图纸、设备表、工程范围、预算和工期。条件冲突时返回主管机关，不自行取舍。

4. 周知、施工和检查准备

完成当地要求的近邻程序，施工变更及时复询；保存竣工图、设备证明、检测、照片和合同。

5. 申请与现场检查

  1. 使用所在地当前申请表和附件清单。
  2. 提交、补正并保持版本控制。
  3. 预约现场检查，记录整改项。
  4. 完成整改并取得许可。

6. 开业交接

  • 核对许可名称、地址、类别、人数和设施。
  • 完成标识、名簿、本人确认、消防、清扫、投诉和财务SOP。
  • OTA和PMS刊登与许可内容一致。
  • 完成员工培训和模拟异常演练。

7. 许可后变更

法人、名称、设施、人数、管理、重大改造、停止或废业前，先确认是变更、重新许可还是其他手续。

时间线：本站不承诺固定申请周期。咨询、设计、施工、补正和检查均取决于物件与管辖机关。
