旅馆业许可申请流程

旅馆业是多部门并行判断。任何一个部门的初步认可，都不等于项目整体已可开业。

复核所在地条例

阶段1：投资前初筛

  1. 确认所有权、租赁、规约和改造权限。
  2. 调查用途地域和是否可作为旅馆使用。
  3. 取得建筑资料并核对现况。
  4. 形成营业类别、人数和管理体制草案。

阶段2：多部门事前咨询

  1. 保健所确认营业类别和当地结构设备标准。
  2. 消防署确认用途、设备、疏散和届出。
  3. 建筑部门确认用途、用途变更和现况问题。
  4. 当地条例要求时完成近邻、学校或其他程序。

阶段3：设计与施工

  1. 由建筑士等把行政意见反映到图纸。
  2. 对消防、卫生、给排水、换气和隔音形成预算。
  3. 施工变更及时返回相关部门确认。
  4. 整理竣工资料、检测和设备证明。

阶段4：申请与检查

  1. 提交许可申请和当地要求附件。
  2. 配合书面审查、补正及其他手续。
  3. 预约并接受现场检查。
  4. 完成整改并取得许可。
  5. 核对许可内容、名称、人数和设施后开业。

阶段5：许可后

  • 持续维护结构设备与卫生标准。
  • 办理变更、停止或废业手续。
  • 保存宿泊者名簿、清扫、消防和事故记录。
