住宅宿泊事业定期报告

以届出住宅为单位，从宿泊者名簿和实际入住记录生成报告，不直接采用平台收入或预订数量代替。

适用住宅宿泊事业
最后核验2026-07-05
复核期限2026-10-03
报告节奏：每年2、4、6、8、10、12月的15日前，报告前两个月的实际宿泊情况。自治体另有表格或提交要求时，以届出地当前说明为准。

报告数据

  • 届出住宅实际让人住宿的日数
  • 宿泊者数
  • 延べ宿泊者数
  • 按国籍划分的宿泊者数

执行步骤

  1. 锁定报告期间，导出对应届出住宅的名簿和入住记录。
  2. 排除取消、未入住和测试订单，核对跨月住宿。
  3. 复核住宿日数、宿泊者数、延べ人数及国籍分类。
  4. 由第二人或负责人核对异常订单和180日累计。
  5. 原则上通过民泊制度运营系统提交；无法使用时确认当地纸面方式。
  6. 保存提交完成画面、回执、使用数据和复核记录。

异常升级

  • PMS、OTA、门锁和名簿人数不一致时不得直接提交。
  • 接近或疑似超过180日、漏报往期、届出编号错误时联系主管窗口。
  • 管理业者掌握的数据不完整时，由住宅宿泊事业者确认责任分工后补齐。

常见错误

  • 把预订间夜当作实际宿泊日数。
  • 多个届出住宅合并报告。
  • 只保存CSV，不保存提交完成证据。
  • 误把住宅宿泊事业报告周期套用于旅馆业。

官方依据

  • 观光厅：住宅宿泊事業者の業務[2]报告期限、项目和提交原则
  • 观光厅：電子宿泊者名簿・定期報告方法电子名簿和报告CSV
