# Minpaku Knowledge

日本民宿・旅馆合规知识库，重点覆盖东京地区的住宅宿泊事业、旅馆业、特区民泊、消防、建筑、卫生、运营与官方资料索引。

## 技术

- 原生 HTML / CSS / JavaScript / JSON
- 无后端、数据库或构建步骤
- GitHub Pages 发布
- 自定义域名：`minpakuk.1211.world`

## 本地预览

在项目根目录启动任意静态 HTTP 服务，然后访问根路径。页面使用根相对链接，不建议通过 `file://` 直接打开。

## 内容边界

本站用于调查、判断和定位官方依据，不构成法律意见，也不代替管辖自治体、保健所、消防署、建筑部门或专业人士对具体物件的判断。

## 校验

运行 `python3 scripts/validate.py` 检查内部链接、数据完整性和复核期限；增加 `--external` 可检查官方外链。
